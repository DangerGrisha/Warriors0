## blue_rose_guardian

Текстуры опциональны — без ресурспака предметы выглядят как базовые ваниль-предметы (роза/перо/краска).

- `Rose Oath` → `/give @s minecraft:iron_sword[minecraft:item_model="lastwar:blue_rose_guardian_rose_oath"]`
- `Blue Rose Ward` → `/give @s minecraft:cornflower[minecraft:item_model="lastwar:blue_rose_guardian_ward"]`
- `Rosebind` → `/give @s minecraft:blue_dye[minecraft:item_model="lastwar:blue_rose_guardian_rosebind"]`
- `Petal Step` → `/give @s minecraft:feather[minecraft:item_model="lastwar:blue_rose_guardian_petal_step"]`
- `Heritage Bloom` → `/give @s minecraft:light_blue_dye[minecraft:item_model="lastwar:blue_rose_guardian_heritage_bloom"]`
- `Garden of the First Rose` → `/give @s minecraft:blue_orchid[minecraft:item_model="lastwar:blue_rose_guardian_garden"]`

Якоря роз (невидимый ArmorStand-шлем) переиспользуют `blue_rose_guardian_ward` и
`blue_rose_guardian_garden` поверх `minecraft:cornflower`. Проще выдавать через `/kitgive BlueRose <oath|ward|rosebind|petal|heritage|garden> [player]`.
