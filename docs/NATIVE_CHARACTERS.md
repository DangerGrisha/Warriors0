# Родные персонажи Warriors0

Документация по «родным» персонажам проекта `Warriors0` (не переносились из Last_Warriors).
Это **компаньон** к [`MIGRATED_CHARACTERS.md`](MIGRATED_CHARACTERS.md) — вместе два документа
покрывают всех персонажей.

Персонажи здесь: **AirWalker, BurgerMaster, DIO, Ganyu, HaoHao, HuTao, Naruto, Opera, Uraraka,
SaberLight, SaberDark**.

- Выдача: `/kitgive <Герой> <предмет> [игрок]` (право `kits.give`, по умолчанию OP)
- Тики → секунды: **20 тиков = 1 секунда**; кулдауны в мс переведены в секунды.
- Числа взяты напрямую из констант листенеров.

> Многие способности гейтятся **PDC-ключом предмета** или **scoreboard-тегом класса**
> (например, Saber — тегами `LightSaber`/`DarkSaber`, HaoHao-перк — тегом `HaoHao`).
> Где тег обязателен — отмечено.

## Система элементальных реакций

Используется Ganyu и HuTao (Pyro). Файл: `org/money/money/combat/ElementalReactions.java`.

На цель вешается «аура» элемента (`PYRO`/`CRYO`/`HYDRO`/`ELECTRO`) как scoreboard-тег на N тиков.
Если ударить другим элементом по уже висящей ауре — урон умножается по таблице реакций
(напр. PYRO по ауре CRYO ×2.0; CRYO по PYRO ×1.5; электро-связки ×1.5 и т.д.), обе ауры
«съедаются», проигрывается SFX. Способности Ganyu наносят **CRYO**; HuTao Pyro — **PYRO**.

## AirWalker
Герой ветра: воздушный якорь и сдув врагов, воздушный слэш, невидимость и ульта с замедленным падением.

### 1. Wind (воздушный поток) — `/kitgive AirWalker wind`
- **Предмет:** `RED_DYE` с именем `Wind`. PDC `airwalker_wind_item`. Заряды = размер стака.
- **Активация:** ПКМ — поставить/переместить якорь; ЛКМ — пустить поток. В Shift поток идёт от игрока, иначе от якоря.
- **Механика:** сдувает живых в радиусе по направлению взгляда (velocity `dir×2.5` + Y `+0.15`), Levitation на ~0.5–1 c. Частицы `GUST` (фолбэк `CLOUD`); звуки ванильные (`BLOCK_AMETHYST_BLOCK_CHIME`, `ENTITY_BREEZE_IDLE_GROUND`). Якорь — невидимый ArmorStand.
- **Урон:** — (отбрасывание + левитация).
- **Кулдаун:** антиспам 0.5 c (10т); тратит 1 заряд, возврат +1 через 20 c. При активной ульте `WindUlt` заряд не тратится.
- **Радиус/дальность:** сдув R5.0; установка якоря до 60 бл (120 при ульте).

### 2. WindUlt (ульта) — `/kitgive AirWalker windult` (`ult`)
- **Предмет:** `RED_DYE` с именем `WindUlt`. PDC `airwalker_wind_ult_item`.
- **Активация:** ПКМ.
- **Механика:** тег `WindUlt` на 45 c. Пока в воздухе — постоянный Slow Falling (amp 15, в Shift amp 5). Усиливает Wind (без расхода зарядов, якорь до 120 бл). Звуки ванильные.
- **Кулдаун:** предмет возвращается через 45 c (окончание ульты).
- **Длительность:** 45 c (900т).

### 3. Wind Sword (воздушный слэш) — `/kitgive AirWalker windsword` (`sword`)
- **Предмет:** `IRON_SWORD` с именем `Wind Sword`. PDC `airwalker_wind_sword`.
- **Активация:** ПКМ — выпускает летящий слэш по взгляду (шаг 0.65 бл/тик).
- **Механика:** бьёт только **вражеских игроков** (по командам); частицы `SWEEP_ATTACK`/`CLOUD`/`CRIT`, звуки ванильные.
- **Урон:** `12.0` (6 сердец; в комментарии кода ошибочно «3 hearts»).
- **Кулдаун:** `7 c` (actionbar-таймер).
- **Дальность:** слэш 15 бл, радиус попадания 1.1.

### 4. Invis (невидимость) — `/kitgive AirWalker invis` (`windinvis`)
- **Предмет:** `RED_DYE` с именем `Invis`. PDC `airwalker_invis_item`.
- **Активация:** ПКМ — Invisibility I, тратит 1 предмет.
- **Кулдаун:** возврат предмета через **180 c (3 мин)** (продолжается даже офлайн).
- **Длительность:** невидимость 30 c.

## BurgerMaster
Повар: гриль для бургеров с эффектами, садовая ферма-платформа, режим «буйства» через объедание.

### 1. Grill (гриль) — `/kitgive BurgerMaster grill`
- **Предмет:** `GRAY_GLAZED_TERRACOTTA` с именем `grill`. PDC `burger_grill_item`.
- **Активация:** поставить блок → спавнится зажжённый `CAMPFIRE` + невидимый ArmorStand. ПКМ по костру владельцем — GUI-меню (сундук 27 слотов).
- **Механика:** в меню 3 рецепта (Meat→Strength I, Diet→Regeneration I, Sea→Speed I), до 4 готовок сразу. Готовый бургер — `PUMPKIN_PIE` (PDC `burger_kind`), при поедании даёт эффект 30 c + звук `ENTITY_PLAYER_BURP`. Сторож раз в 1 c: если гриль исчез — предмет возвращается владельцу.
- **Кулдаун:** возврат предмета через **60 c (1 мин)** после установки (как у Garden); плюс предмет возвращается при разрушении гриля.
- **Длительность:** готовка 1 бургера — **30 c**; эффект бургера — 30 c.

### 2. Garden Platform — `/kitgive BurgerMaster garden`
- **Предмет:** `ORANGE_GLAZED_TERRACOTTA` с именем `Garden Platform`. PDC `garden_platform_item`.
- **Активация:** поставить блок → строится ферма 5×5 (грядки с пшеницей, вода, забор 7×7, колонна: 2× верстак + коптильня).
- **Кулдаун:** возврат предмета через **60 c (1 мин)**.
- **Длительность:** постройка постоянная (не удаляется автоматически).

### 3. Hungry master (буйство) — `/kitgive BurgerMaster hungry`
- **Предмет:** `RED_DYE` с именем `Hungry master`. PDC `hungry_master_item`.
- **Активация:** ПКМ — Hunger X на 5 c, обнуляет голод; открывает окно 10 c «съешь 3 бургера».
- **Механика:** успел съесть 3 → режим «БУЙСТВО» 60 c: Resistance X, Speed IV, Weakness VI, частицы `ANGRY_VILLAGER`, звук `ENTITY_WITHER_SPAWN`.
- **Кулдаун:** при успехе ~120 c, при провале ~60 c (возврат предмета).
- **Длительность:** окно 10 c; зверь 60 c.

### 4. Burger Sword — `/kitgive BurgerMaster sword`
- **Предмет:** `IRON_SWORD` с Knockback II (создаётся прямо в `KitGiveCommand`), ванильное поведение.

## DIO
Вампир-станд: «рука» (Za Warudo) для рывка и серии ударов, остановка времени, вампирская форма с лайфстилом.

### 1. Hand (Za Warudo) — `/kitgive dio hand`
- **Предмет:** `DIAMOND_SWORD` с именем `hand`. PDC `dio_hand_item`. При держании рядом спавнится невидимый станд-ArmorStand, следующий за игроком.
- **Активация:** ПКМ — рывок станда вперёд (шаг 0.8 бл/тик).
- **Механика:** при столкновении со стеной/врагом/макс.дистанцией — «якорится» на 3 c: серия ударов раз в 0.3 c по ближайшему врагу + Slowness I, гасит нокбэк жертвы. Цель — только вражеские игроки. Звук — кастомный `my_sounds.muda`.
- **Урон:** обычный удар мечом-«рукой» — **3.0** (зафиксировано через `EntityDamageByEntityEvent`; было ванильных 7 у алмазного меча). Серия ударов станда в якоре — `5.0` за удар («истинный», снимает абсорбцию; фолбэк мин. 2.0).
- **Кулдаун:** — (ограничен длительностью якоря).
- **Радиус/дальность:** dash до 6 бл; удар R1.3; якорь 3 c.

### 2. TIME_STOP — `/kitgive dio timestop`
- **Предмет:** `RED_DYE` с именем `TIME_STOP`. PDC `dio_time_stop`.
- **Активация:** ПКМ (тратит 1).
- **Механика:** заводка 3.9 c (мигающая слепота/тьма всем кроме владельца), затем стоп времени **5 c**: заморозка всех игроков (отмена движения/кликов/атак), пробует ванильный `/tick freeze`. Звук старта — кастомный `my_sounds.dio1`.
- **Урон:** —.
- **Кулдаун:** **120 c (2 мин)** (возврат предмета).

### 3. Vampire — `/kitgive dio vampire`
- **Предмет:** `RED_DYE` с именем `Vampire`. PDC `vampire_item`.
- **Активация:** ПКМ (тратит 1).
- **Механика:** заводка 3 c (Slowness V + звуки), затем форма **40 c**: +20 макс. HP, мгновенный хил +10 HP, маска-шлем (снять нельзя), **лайфстил 36%** от урона по игрокам.
- **Кулдаун:** **180 c (3 мин)** (возврат предмета по окончании формы).

> `DioStandFollower` (пассивный авто-станд по тегу `DIO`) **отключён** в `Main.java` (закомментирован).

## Ganyu
Крио-лучница на элементальных реакциях (наносит **CRYO**).

### 1. Ganyu Bow (ледяная стрела) — `/kitgive ganyu bow`
- **Предмет:** `BOW` с Infinity. PDC `ganyu_bow`.
- **Активация:** натянуть ≥3 c и выстрелить — стрела «ледяная» (PDC `ganyu_icy_arrow`).
- **Механика:** при попадании — ледяной взрыв (частицы DUST/SNOWFLAKE/CLOUD, звуки стекла/аметиста), всем целям SLOWNESS II на 2.5 c + фриз. Элемент CRYO.
- **Урон:** прямое попадание базово `1.0` (+урон стрелы), AOE `6.0` — всё через множитель реакции.
- **Кулдаун:** — (заряд 3 c; Infinity не тратит стрелы).
- **Радиус:** AOE R3.0.

### 2. Frostbud (крио-бутон) — `/kitgive ganyu bud`
- **Предмет:** `CYAN_DYE` `§bFrostbud` (PDC `ganyu_bud`); после установки — `ENDER_EYE` `§fView Changer`.
- **Активация:** ПКМ Frostbud — поставить бутон; ПКМ View Changer — развернуть взгляд врагов на бутон.
- **Механика:** невидимый ArmorStand-бутон живёт 20 c, каждые 3 c пульс: крио-кольца (R10), урон + SLOWNESS II + фриз по целям. View Changer разворачивает врагов в R10.
- **Урон:** `4.0` за пульс (через реакцию).
- **Кулдаун:** **80 c** (возврат Frostbud после смерти бутона).

### 3. Everfrost Core (IceSphere — ульта) — `/kitgive ganyu ult` (`core`)
- **Предмет:** `HEART_OF_THE_SEA` `§bEverfrost Core`. PDC `ganyu_ult_item`.
- **Активация:** ПКМ (тратит 1), звук `ITEM_TRIDENT_RIPTIDE`.
- **Механика:** в небе сфера-ArmorStand (круг R30), «ледяной дождь» — падающие хиттеры, при ударе AOE по врагам (частицы/звуки льда, SLOWNESS II + фриз).
- **Урон:** `4.0` за удар хиттера (через реакцию). AOE удара R6.
- **Кулдаун:** **150 c (2.5 мин)**. Длительность ульты ~50 c.

## HaoHao
«Король» ближнего боя: меч с рывком-cleave и возвратом урона, маска-отражение, командные пассивки.

### 1. King's Sword — `/kitgive HaoHao swordshield` (`shield`/`sword`)
- **Предмет:** `DIAMOND_SWORD` `King's Sword`. PDC `haoh_king_sword`. Вспом.: `WOODEN_SWORD` «HandR/HandG/HandY».
- **Активация:** ПКМ — рывок+cleave; Shift+ПКМ — режим щита (HandR); ПКМ по HandG — возврат урона.
- **Механика:** рывок (velocity ×1.35) → через 3т AoE-cleave в конусе. Пассивно: 25% шанс наложить Glowing. Щит: получив удар (в руке HandR/HandG), предмет → HandG на 1 c; активация возвращает атакующему его урон и лечит носителя на 50%.
- **Урон:** cleave `6.0` (отброс 0.8); возврат = полученный урон.
- **Кулдаун:** рывок `7 c`; после возврата урона рука = HandY на 10 c.

### 2. Mask — `/kitgive HaoHao mask`
- **Предмет:** `YELLOW_DYE` `Mask` (PDC `mask_yellow`); включённая — `GREEN_DYE` `Mask` (PDC `mask_green`).
- **Активация:** ПКМ — переключение жёлтая↔зелёная (тег `MaskGreen`).
- **Механика:** при активной зелёной маске урон между вражескими игроками **отражается** обратно атакующему через 1 c (не отражается смертельный удар и по тиммейтам).
- **Кулдаун:** — (свободное переключение).

### 3. Командные пассивки (HaoHaoPerkListener) — пассивка
- **Гейт:** scoreboard-тег `HaoHao` у «короля». Предмета нет.
- **Механика:** после удара короля (окно 4 c):
  - враг сработал тотемом → союзникам в R10 Strength II на 10 c;
  - король убил игрока → тиммейтам Strength III на 10 c;
  - король умер → тиммейтам Strength II на 30 c.

## HuTao
Пиро-боец ближнего боя (наносит **PYRO**).

- **Предмет:** `ORANGE_DYE` `§6Pyro Status`. PDC `hutao_pyro_status`.
- **Активация:** ПКМ (тратит 1) — «пиромод» 10 c.

### 3. BOOM (ульта) — `/kitgive HuTao ult`

- **Урон:** `28.0` (через PYRO-реакцию), поджог целей; задевает и тиммейтов.
- **Кулдаун:** **120 c (2 мин)**. Радиус взрыва `10.0`.

## Naruto
Мобильный боец: Расенган, телепорт-исчезновение, теневые клоны.

### 1. Rasengan — `/kitgive Naruto rasengan`
- **Предмет:** `BLUE_DYE` `Rasengan` (PDC `naruto_rasengan`) → при зарядке `HEART_OF_THE_SEA` «Rasengan (charged)».
- **Активация:** ПКМ — заряд (Jump Boost III на 15 c); затем ЛКМ по блоку / удар «ядром» по сущности — взрыв-нокбэк.
- **Механика:** AoE-нокбэк (включая владельца, кап скорости 2.8), частицы `CLOUD`/`END_ROD`, иммунитет к фолл-урону 2 c после удара.
- **Урон (по сущности):** `10.0 + 1.2 × блоки падения`, макс. `70`. По блоку — только нокбэк.
- **Кулдаун:** **25 c** (возврат красителя). Ядро живёт 15 c.

### 2. Disappearance Technique — `/kitgive Naruto disappear`
- **Предмет:** `RED_DYE` `Disappearance Technique`. PDC `disappearance_technique`.
- **Активация:** ПКМ (тратит 1) — дым + звук, через 2 c телепорт по взгляду.
- **Дальность:** до 25 бл.
- **Кулдаун:** **60 c**.

### 3. Clones — `/kitgive Naruto clones`
- **Предмет:** `YELLOW_DYE` `clones`. PDC `naruto_clones`.
- **Активация:** ПКМ (тратит 1) — спавн 15 зомби-клонов (1 HP, золотая броня «Naruto») вокруг точки взгляда (до 20 бл).
- **Механика:** клоны не таргетят владельца/тиммейтов; их удар даёт Slowness на 1 c.
- **Кулдаун:** **90 c (1.5 мин)** (возврат предмета + диспавн клонов).

## Opera
Поддержка/мобильность: трансформация в коня для команды, командная аура, пассивная Скорость.

### 1. Transformation (конь) — `/kitgive Opera transformation`
- **Предмет:** `RED_DYE` `transformation`. PDC `opera_transformation`.
- **Активация:** ПКМ — toggle вкл/выкл.
- **Механика:** спавнит привязанного коня (следует сбоку), владелец получает Invisibility, Weakness, Jump Boost III, Speed IV (бесконечно до выкл). Тиммейт может сесть; враг/владелец — нет. Урон по коню переходит на владельца; конь не умирает, пока жив владелец.
- **Кулдаун:** — (toggle).

### 2. Opera Aura — `/kitgive Opera aura`
- **Предмет:** `PURPLE_DYE` `Opera Aura`. PDC `opera_aura_item`.
- **Активация:** ПКМ — GUI (режимы Attack/Formation/March/Off).
- **Механика:** каждые 0.5 c накладывает тиммейтам в R10 эффект на 1.5 c: Attack→Strength I, Formation→Resistance I, March→Speed I. Кольцо частиц DUST.
- **Кулдаун:** —.

### 3. Пассивка Opera — пассивка
- **Гейт:** тег `Opera`. Каждые 2 c всем с тегом `Opera` — Speed I на 3 c.

## Uraraka
Гравитационный персонаж: молот-оглушение, окно левитации, лечащий пост, метка-проклятие.

### 1. Hammer (Glove) — `/kitgive uraraka glove`
- **Предмет:** `DIAMOND_SWORD` `Hammer`. PDC `uraraka_glove`.
- **Активация:** пассивка при **критическом** ударе мечом.
- **Механика:** крит даёт цели Slowness II на 1 c, звук `ENTITY_PLAYER_ATTACK_CRIT`, частица `SWEEP_ATTACK`.
- **Урон:** — (свой урон не добавляет). Радиус 1.5.

### 2. Gravity — `/kitgive uraraka gravity`
- **Предмет:** `FEATHER` `gravity` (PDC `uraraka_gravity`); при активации даётся `FEATHER` `CancelGravity`.
- **Активация:** ПКМ — старт (заводка 3 c со Slowness III), затем окно **10 c**.
- **Механика:** в окне любой удар владельца по живой цели даёт ей LEVITATION I на 8 c (можно поднять много целей). ПКМ по `CancelGravity` — досрочная отмена (снимает левитации).
- **Кулдаун:** **120 c (2 мин)** (возврат предмета).

### 3. Heal Post — `/kitgive uraraka healpost`
- **Предмет:** `RED_GLAZED_TERRACOTTA` `Heal Post`. PDC `uraraka_heal_post`.
- **Активация:** поставить блок — спавнится якорь-ArmorStand.
- **Механика:** каждые 5 c лечит тиммейтов в R10 на **+4 HP** (частица HEART). Сторож следит за блоком.
- **Кулдаун:** — (пока стоит блок).

### 4. Levitation Mark — `/kitgive uraraka levmark`
- **Предмет:** `NETHER_STAR` `Levitation Mark`. PDC `levitation_mark`.
- **Активация:** ПКМ — «зарядка» (тратит 1), затем следующий удар по игроку ставит метку на 60 c.
- **Механика:** когда отмеченную цель кто-то бьёт — срабатывает: если бил владелец, он получает разовый жёсткий подброс (LEVITATION ур.30 на 1 c); если другой — на атакующего вешается «проклятие» LEVITATION II цепочкой ~10 c.
- **Кулдаун:** **120 c (2 мин)** (возврат предмета).

## SaberLight / SaberDark
Дуэлянты с мечом-щитом «Excalibur» и ультой-лучом «Soul Release». Light и Dark идентичны механически,
отличаются темой (Light — золотой; Dark — чёрно-красный) и именами (`ExcaliburD`, `Soul ReleaseD`).
Подробности: [`SABER_PLAYER_GUIDE.md`](SABER_PLAYER_GUIDE.md), [`SABER_TECHNICAL_REFERENCE.md`](SABER_TECHNICAL_REFERENCE.md).

> **Обязательны теги класса:** `LightSaber` / `DarkSaber`. Без тега способности и UI щита не работают.

### 1. Excalibur (меч-щит) — `/kitgive SaberLight excalibur` / `/kitgive SaberD excaliburd`
- **Бой:** базовый урон `7.0`, `+0.5` за душу; скорость атаки тяжелеет `−0.1`/душа (мин `0.4`).
- **Блок (guard):** при корректном блоке щит держит `80%`, пропускает `20%`. Порог пробития — `12.0`
  заблокированного урона; при пробитии — стан `5 c`, затем полное восстановление. Реген щита `+1.0` каждые `2 c`.
- **Души:** копятся за тотем-поп от удара Excalibur и за килл от удара Excalibur; усиливают оружие.

### 2. SoulTrades (магазин душ) — `/kitgive SaberLight soultrades`
- **Предмет:** синий краситель, открывает меню `Soul Contracts` (работает для обоих классов).
- **Покупки:** Тотем (первый 3 души, далее 4); Перм. Speed +1 (3 души, макс III); Врем. Jump X на 10 c (1 душа); Перм. Jump +1 (2 души, макс III).
- Добавить души: `/kitgive SaberLight souladd [кол-во] [игрок]`.

### 3. Ult — Soul Release / Soul ReleaseD — `/kitgive SaberLight ult` / `/kitgive SaberD ultd`
- **Стоимость:** `4` души. **Кулдаун:** `25 c`.
- **Тайминг каста:** луч начинает подниматься на `8.0 c`, максимум + вспышка на `9.5 c`, частицы рассеивания на `14.5 c`, фаза выстрела с `17.5 c`.
- **Прицел/движение:** во время заряда камера свободна; в начале выстрела направление фиксируется по взгляду; во время выстрела движение заблокировано.
- **Урон:** основное давление — от движущихся «мин»-импактов луча; финальный взрыв в основном кинематографичен + нокбэк (не главный килл).
